## 2026-07-02 - 인쇄 미리보기 창 추가

### 요청 원문

> Xcapture 프린트 하면 2번째 첨부 이미지 같이 단순한 인쇄창이 나오는데.  인쇄창에 첫번째 같이  미리보기가 나오도록 할수있으면 해줘. 꼭 첫번째 이미지 같이는 안되더라도.. 용지 사이즈에 맞게 어떻게 프린트 되는지 보여주고, 가로,세로 정도 설정해서. 프린트할수있도록 해줘.

### 결과 요약

- 기존 `PrintDialog` 직접 호출 대신 XCapture 전용 인쇄 미리보기 창을 추가했다.
- 미리보기 창에서 세로/가로 방향, A4/Letter 용지, 페이지 맞춤/원본 크기 출력 방식을 선택할 수 있게 했다.
- 미리보기에서 보이는 배치와 실제 인쇄 시 이미지 배치 계산을 같은 로직으로 맞췄다.
- 버전을 `1.0.16`으로 올리고 설치파일 `Installer/XCaptureSetup.exe`를 다시 생성했다.
